# 🚀 FUTURE_DS_01 - Business Sales Analysis

## 📊 Project Title
Business Sales Performance Analytics

---

## 🎯 Objective
To analyze business sales data and extract meaningful insights such as total sales, region-wise performance, category-wise performance, and top-selling products.

---

## 🛠 Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook / VS Code  

---

## 📁 Dataset Description
The dataset contains sales transaction details including:
Order ID, Order Date, Customer Name, Product Name, Category, Region, and Sales Amount.

---

## 📊 Key Findings

- 💰 Total Sales: 2,261,536.78  
- 📍 Best Performing Region: West  
- 📦 Best Category: Technology  
- 🏆 Top Selling Product: Canon imageCLASS 2200 Advanced Copier  

---

## 📈 Visualizations
- Region-wise Sales Graph  
- Category-wise Pie Chart  
- Top 10 Products Bar Chart  

*(All graphs are included in the PDF report)*

---

## 💡 Recommendations
- Focus more on high-performing regions like West  
- Increase investment in Technology category  
- Maintain stock of top-selling products  
- Improve sales in lower-performing regions  

---

## 🏁 Conclusion
This project provides insights into business sales performance and helps in better decision-making to improve revenue and efficiency.
